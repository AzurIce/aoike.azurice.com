「基础算法」归并排序

算法核心

分治算法：

分别排序左右，再合并

关键 - 合并 - 双指针 + 辅助数组

原数组 -> 辅助数组 -> 原数组

Code

int tmp[MAXN];
void mergeSort(int *a, int l, int r) {
	if (l == r) return;
	
	int mid = l + r >> 1;
	mergeSort(a, l, mid);
	mergeSort(a, mid+1, r);
	
	int t = 0, i = l, j = mid+1;
	while (i <= mid && j <= r) {
		if(a[i] < a[j]) {
			tmp[t++] = a[i++];
		} else {
			tmp[t++] = a[j++];
			// cnt += mid-i+1;
		}
	}
    while (i <= mid) tmp[t++] = a[i++];
	while (j <= r)   tmp[t++] = a[j++];
	
	for (int i = 0; i < t; i++) a[l+i] = tmp[i];
}
