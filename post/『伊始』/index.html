[Manjaro KDE]

〇、配置国内源、库

1. 配置 pacman-mirrors

官方 Pacman-mirrors Wiki文档：Pacman-mirrors - Manjaro

pacman-mirrors 命令可以用来生成镜像源列表，并写入 /etc/pacman.d/mirrorlist 中。

切换镜像源至中国

sudo pacman-mirrors -c China

-c 是 --country 的简写

2. 配置 manjarocn 库

编辑 /etc/pacman.conf ，添加以下内容：

[manjarocn]
Server = https://repo.manjarocn.org/stable/x86_64

导入 GPG keys：

sudo pacman-key --recv-keys 974B3711CFB9BF2D && sudo pacman-key --lsign-key 974B3711CFB9BF2D

3. 更新软件源

sudo pacman -Syyu

1. 配置中文输入法

fcitx5输入法框架
sudo pacman -S fcitx5-im

fcitx5-im 中提供了 fcitx5 本体、配置工具、和必要的 输入法模块。

安装完成后，需编辑 ~/.pam_environment 文件，设置以下环境变量：

GTK_IM_MODULE DEFAULT=fcitx
QT_IM_MODULE  DEFAULT=fcitx
XMODIFIERS    DEFAULT=@im=fcitx
INPUT_METHOD  DEFAULT=fcitx
SDL_IM_MODULE DEFAULT=fcitx

fcitx5 仅提供基本框架，只支持英文输入，若要进行其它语言的输入还需安装 输入法引擎 。

安装 fcitx5-rime 输入法引擎：

sudo pacman -S fcitx5-rime

随后配置 cloverpinyin 输入方案：

参照官方文档 fkxxyz/rime-cloverpinyin: 🍀️四叶草拼音输入方案，做最好用的基于rime开源的简体拼音输入方案！ (github.com)

9. 雷蛇驱动配置软件

到 pamac 程序中，首选项 -> 高级 -> AUR 中启用AUR支持。

搜索 openrazer-meta并将其安装（应该会自动安装 openrazer-driver-dkms, openrazer-daemon, python-openrazer）几个包。

在终端中输入

sudo gpasswd -a $USER plugdev
reboot

重启后验证是否有问题：

  1. 显示 Kernel module 日志

    sudo dmesg

  2. 显示 razer_mount 日志

    sudo journalctl -b | grep -i razer_mount

  3. 显示 Daemon 日志

    cat ~/.local/share/openrazer/logs/razer.log

  4. 验证 Kernel modules 是否正确构建

    sudo dkms install openrazer-driver/$version

    $version 改为安装的版本号，我这里是3.1.0-1 故改为 3.1.0

    若输出 ... already installed on ... 则正常

    若输出 Your Kernel headers for kernel xxx cannot be found at ... 则缺少 kernel headers

    按照提示的版本，进行安装

    sudo pacman -S linux513-headers

    我的内核是 5.13.12-1 故写 linux513

    若是 4.19 内核则写 linux419

  5. 验证 Kernel modules 是否能正确加载

    sudo modprobe razerkbd

    若无输出，则为正常。

附上官方 github wiki 上的 troubleshooting 页面：Troubleshooting · openrazer/openrazer Wiki (github.com)

随后搜索 razergenie 并将其安装，并重启（有问题就重启试试），可以进行正常配置。
