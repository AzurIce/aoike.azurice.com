一、激活

使用16进制编辑工具编辑 sublime_text.exe 。

这里我使用了一个在线工具 HexEd.it - Browser-based Online and Offline Hex Editing

搜索第一个 80 38 00 并用 FE 00 90 将其替换。

二、必备

  1. 安装 Package Control

    打开指令面板，执行 Install Package Control 。

    打开指令面板快捷键：Ctrl + shift + p

    或直接 Tools -> Install Package Control...

    随后就可以使用命令面板的 Install Package: ... 进行 Package 的安装。
