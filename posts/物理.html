40. 光的折射 全反射

一、 折射率

  1. 定义式

    \Large{ \begin{array}{lcc} n &=& \frac{\sin i_{真空}}{\sin r_{介质}}\\ n_{12} &=& \frac{\sin\theta_1}{\sin\theta_2} \end{array} }

  2. 计算式

    n=cvn=λ0λ\Large{ \begin{array}{ccc} n &=& \frac{c}{v}\\ n &=& \frac{\lambda_0}{\lambda}\\ \end{array} } ​n​n​​​​=​=​​​​v​​c​​​​λ​​λ​0​​​​​​

v<cv<cv<c 故任何介质折射率 >1>1>1

光疏(nnn 小) →\to→ 光密(nnn 大)：入射角 >>> 折射角，折射率 >1>1>1

光密(nnn 大) →\to→ 光疏(nnn 小)：入射角 <<< 折射角，折射率 <1<1<1

二、全反射

光密 →\to→ 光疏，折射角 =90∘= 90^\circ=90​∘​​ ，入射角 ≥\geq≥ 临界角。

\Large{ n_{密疏} = \frac{\sin i}{\sin r} = \frac{\sin C}{\sin 90} = \sin C\\ n_{密疏} = \frac{1}{n_{疏密}}\\ \sin C = \frac{1}{n_{疏密}} \approx \frac{1}{n} }

三、色散

频率高的光折射率大（折射对频率大的光影响大）

红光 →\to→ 紫光 频率增大。
