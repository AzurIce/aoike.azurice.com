Local Variable Storage Classes

存储类别

auto
static
extern
register

auto 只能用在函数中。

局部变量只能是 auto、static 或 register 。

  • register 存在 CPU寄存器中的 auto 变量

    若无寄存器则为 auto。

  • auto 是默认的类型

    动态存储区

    短生命周期变量。自动申请、释放。

    is automatically reserved each time a function declaring automatic variables is called

    As long as the function has not returned control to its calling function, all automatic variables local to the function are alive

  • static

    静态存储区

    长生命周期变量。从程序运行开始到整个程序结束始终存在。

    在程序被初次编译的时候被初始化，有些编译器在第一次定义语句执行的时候初始化。

    depend on compiler

全局变量：

  • static 只能在当前文件中使用

  • extern 引用其他文件的变量

    可不写变量类型
