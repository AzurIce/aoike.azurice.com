DrawableHelper 中的一些方法

以下 color 均为 24位整数（0xFFFFFFFF）ARGB

  1. fill(MatrixStack matrices, int x1, int y1, int x2, int y2, int color)

    用颜色 color 填充 (x1, y1) 到 (x2, y2) 的矩形区域。

    ![fill](V:_Posts\Halo\Minecraft\Fabric\Mojang's code - GUI #1 Screen.assets\fill.gif)

  2. drawHorizontalLine(MatrixStack matrices, int x1, int x2, int y, int color)

    在纵坐标为 y 处，从 x1 到 x2 绘制一条颜色为 color 的水平直线。

    ![drawHorizontalLine](V:_Posts\Halo\Minecraft\Fabric\Mojang's code - GUI #1 Screen.assets\drawHorizontalLine.gif)

  3. drawVerticalLine(MatrixStack matrices, int x, int y1, int y2, int color)

    在横坐标为 x 处，从 y1 到 y2 绘制一条颜色为 color 的竖直直线。

  4. drawCenteredText(MatrixStack matrices, TextRenderer textRenderer, String text, int centerX, int y, int color)

    String text 也可为 Text text

    在纵坐标为 y 处，绘制颜色为 color 内容为 text 关于 centerX 居中的文字。

    ![drawCenteredText](V:_Posts\Halo\Minecraft\Fabric\Mojang's code - GUI #1 Screen.assets\drawCenteredText.gif)

  5. drawStringWithShadow(MatrixStack matrices, TextRenderer textRenderer, String text, int x, int y, int color)

    drawTextWithShadow(MatrixStack matrices, TextRenderer textRenderer, Text text, int x, int y, int color) 同理。

    在 (x, y) 处绘制颜色为 color 内容为 text 的文字。
