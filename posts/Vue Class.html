一、Vue单文件组件

一个 *.vue 文件。

由三种顶层语法块组成：<template>、<script>、<style> 以及可选的附加自定义块。

<template>
  <div class="example">{{ msg }}</div>
</template>

<script>
export default {
  name: 'HelloWorld',
  data() {
    return {
      msg: 'Hello world!'
    }
  }
}
</script>

<style>
.example {
  color: red;
}
</style>
  • <template> 部分定义组件的模板
  • <script> 部分应该默认导出一个 Vue组件选项对象。
  • <style> 部分定义与组件关联的 CSS

1. Data

组件的 data 选项是一个函数，返回一个对象，Vue会通过响应性系统将其包裹起来以 $data 的形式存储在组件实例中。

export default {
	name: 'HelloWorld',
    data() {
        reutrn {
            msg: 'Hello World!'
        }
    }
}

使用 Vue Class Component 来书写：

@Component
export default class Helloworld extends Vue {
	msg = 'Hello World!'
}

类的属性即为 data 。

注意如果初始值为 undefined 则此类属性不会具有响应性。

为了避免可以使用 null 或 data hook 来代替

@Component
export default class HelloWorld extends Vue {
  // `message` will be reactive with `null` value
  message = null

  // See Hooks section for details about `data` hook inside class.
  data() {
    return {
      // `hello` will be reactive as it is declared via `data` hook.
      hello: undefined
    }
  }
}

2. Methods

组件的 methods 选项是一个包含所需方法的对象，this 指向组件实例。

export default {
	name: 'HelloWorld',
    data() {
        reutrn {
            msg: 'Hello World!'
        }
    },
    methods: {
    	changeText() {
    		this.msg = 'NB!'
    	}
    }
}

使用 Vue Class Component 来书写：

@Component
export default class Helloworld extends Vue {
	msg = 'Hello World!'
    
    changeText() {
        this.msg = 'NB!'
    }
}

类的方法即为 Methods
