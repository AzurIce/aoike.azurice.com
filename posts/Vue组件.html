二、Vue组件

组件是带有名称的可复用实例，在创建好的项目中的 /src/components/ 目录下有这样一个文件 HelloWorld.vue：

<template>
	...
</template>

<script>
export default {
  name: "HelloWorld",
  ...
};
</script>

...

它就是一个组件。

在 App.vue 中是这样使用的：

<template>
  ...
  <HelloWorld ... />
</template>

<script>
import HelloWorld from "./components/HelloWorld.vue";

export default {
  name: "App",
  components: {
    HelloWorld,
  },
};
</script>

...

将组件在 components 中注册后，便可以当作以一个元素来使用。

1. Prop

Prop 是我们可以在组件上注册的一些自定义 attribute。

还是上面的 Helloworld 组件，src/components/Helloworld.vue 中：

<template>
	...
	<h1>{{ msg }}</h1>
	...
</template>

<script>
export default {
  name: "HelloWorld",
  porps: {
  	msg: String,
  },
};
</script>

...

src/App.vue 中：

<template>
  ...
  <HelloWorld msg="Welcome to Your Vue.js App" />
</template>

...
