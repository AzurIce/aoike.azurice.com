Kotlin —— 基本类型

一、数字

  1. 整数

    Byte Short Int Long // 8 16 32 64 位

  2. 小数

    Float Double

    以小数初始化的变量，编译器会推断为 Double 类型。

    如需将一个值显式指定为 Float 类型，请添加 f 或 F 后缀。

字面常量

  • 整数

    • 十进制 123

      Long 类型加 L 后缀：123L

    • 十六进制 0x0F

    • 二进制 0b00001011

  • 小数 默认 Double 123.5 123.5e10

    Float用 f 或 F 标记：123.5f
