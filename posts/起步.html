起步

一、安装 nodejs

二、安装一些必要库

  1. Yarn

    npm install yarn -g

    Yarn 是一个包管理器，npm 也是一个包管理器，可以将 yarn 理解成重新设计的 npm。

    有关二者之间的关系与联系，请参考这篇文章 （这篇文章还没写）。

    要注意将 yarn 的 bin 目录加入到系统环境变量。

  2. @vue/cli

    @vue/cli 是 Vue 的项目脚手架工具，可以用以快速创建项目。

    yarn global add @vue/cli

    可以使用 vue --version 来查看其版本。

    未来可以使用

    yarn global upgrade --latest @vue/cli

    来升级

三、创建项目

进入到要放置项目文件夹的目录，使用以下命令来创建一个 vue 项目：

vue create azur-mc-helper

设置项目相关配置。

四、引入 electron

进入到项目文件夹

vue add electron-builder

五、开始吧！
